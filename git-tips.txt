git stash list --pretty=format:"%C(red)%h%C(reset) - %C(dim yellow)(%C(bold magenta)%gd%C(dim yellow))%C(reset) %<(70,trunc)%s %C(green)(%cr) %C(bold blue)<%an>%C(reset)"
Viser stash liste med id og tid siden (%ci i stedet for %cr gir dato/tid)
git stash apply 2af750cee182252a83cc0d16b0a91a94f7220337  (henter tilbake gammel/mistet stash  SHA-1 koden kan finnes med TortoseGit stash list)

git stash apply	(henter tilbake siste stash).
git stash pop (henter tilbake siste stash og SLETTER). 
Eks. lagre for å se på annen branch før man går tilbake: git stash, git checkout otherbranch, git checkout branch, git stash pop  (evt. git stash apply  for å ta vare på stash, som evt. slettes med git stash drop)
Eks. merge data fra annen branch:  git stash, git checkout branch2, git pull, git stash pop (sjekk rekkefølgen i dette)
